

# Contact

## Issues

If you encounter any issues during use or have any suggestions, please create a new Issue at {{ '[AimRT Official Git Repository]({})'.format(code_site_url) }}. Before creating a new Issue, please search existing Issues to ensure your question hasn't been raised before. When creating a new Issue, please refer to the corresponding template type.

### Bug Report
- **Title**: [Bug] xxxx issue at xxxxx
- **Content**:
  - AimRT version used
  - Environment information including OS, compiler, CMake version, etc.
  - Steps to reproduce
  - Expected behavior
  - Actual behavior
  - Logs or additional materials when the issue occurred

### Feature Request
- **Title**: [Feature] Implement xxxx functionality
- **Content**: Detailed description of usage scenarios and requirements

## Pull requests

We welcome developers to submit code contributions to improve AimRT. You can Fork the AimRT repository under your account, then push a Merge Request to the AimRT project. MRs will be merged into the main branch after passing CI pipelines and administrator reviews.
